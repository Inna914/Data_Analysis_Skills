# Modern Apprenternship 2023-2026 Analysis

## Project Overview

This project features a data analysis report and visual infographics examining trends in modern apprenticeship schemes between the 2023/24 and 2025/26 academic years. Using Power BI for robust data visualization and Canva to design the final, presentation-ready infographics, this project highlights shifts in overall intake and demographic breakdowns based on data from Skills Development Scotland.

## Dashboard Features
The Power BI report includes several key visualizations to explore the dataset interactively:
- **Bar Chart**: Displays the sum of apprentices categorized by year and gender.
- **Pie Chart**: Illustrates the proportional breakdown of apprentices by year.
- **KPI Cards**: Shows the average number of apprentices (4.01K) and the standard deviation (3.15K).
- **Interactive Slicer**: Allows users to filter the dashboard by specific academic years.

## Key Findings
Analysis of the data reveals several notable trends regarding places on modern apprenticeship schemes between 2023 and 2026:
- **Overall intake**: A significant decline in the total number of work-based training places is observed in the 2025/2026 academic year.
- **Peak and Drop**: The number of apprentices peaked in the 2024/25 academic year (12,146), before falling to 11,769 in 2025/26.
- **Gender inequality**: A consistent gender gap persists throughout the observation period. The number of male trainees consistently exceeds the number of female trainees. For example, in the 2025/26 academic year, there were 7,500 male trainees and 4,100 female trainees.
- **Undisclosed/Non-binary Data**: The participation of those who ‘prefer not to say/identify differently’ remains consistently low, at approximately 100 people throughout the entire period.
