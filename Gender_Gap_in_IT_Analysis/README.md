# The Gender and Interest Gap in Digital Apprenticeships

This work demonstrates how data can be cleaned, structured, summarised, and visualised to provide meaningful insights.

I have analysed data on school-based apprenticeships (Foundation Apprenticeships) from recent years. The figures speak for themselves: we risk losing an entire generation of talent.

There is a critically low number of women in software development (just 12%), although they make up the majority in creative media. Overall interest in STEM subjects in schools has fallen by 37% over the past four years.
The total number of students in STEM subjects is falling. Between 2019 and 2023, intake has fallen by almost 40%. This is a ‘wake-up call’ for the entire industry.
